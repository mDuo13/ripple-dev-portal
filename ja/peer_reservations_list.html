peer_reservations_list

[ソース]

peer_reservations_listメソッドは、ピアリザベーションを一覧表示します。

peer_reservations_listメソッドは、権限のないユーザーは実行できない管理メソッドです。

要求フォーマット

要求フォーマットの例:

  • WebSocket
  • JSON-RPC
  • コマンドライン
{
  "id": "peer_reservations_list_example_1",
  "command": "peer_reservations_list"
}

{
    "method": "peer_reservations_list"
}

#Syntax: peer_reservations_list
rippled peer_reservations_list

この要求にはパラメーターは含まれません。

この応答は標準フォーマットに従っており、正常に完了した場合は結果に次のフィールドが含まれます。

Field 型 説明
reservations 配列 既存のピアリザベーションのリスト。各メンバーはピアリザベーションオブジェクトです。詳細は以下のとおりです。

ピアリザベーションオブジェクト

reservations配列の各メンバーは、1つのピアリザベーションを表すJSONオブジェクトです。このオブジェクトのフィールドを次に示します。

Field 型 説明
node 文字列 この予約の対象となるピアサーバーのノード公開鍵（base58）。
description 文字列 （省略される場合があります） このピアリザベーションの説明（ある場合）。
